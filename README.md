## What is QuanticoDB ?
QuanticoDB is a fast, simple, extensible NOSQL database + Artificial Intelligence for PHP.

> <b>require 'Quantico.php';</b>
<br>
<br>

## Requirements
QuanticoDB requires PHP 5.4 or greater.
