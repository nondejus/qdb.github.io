## What is QuanticoDB ?
QuanticoDB is a fast, simple ( 4 command ), extensible NOSQL database + Artificial Intelligence for PHP.

> <b>require 'Quantico.php';</b>
<br>
Qdb::in()<br>
Qdb::out()<br>
Qdb::del()<br>
Qdb::ver()<br>
<br>

## Requirements
QuanticoDB requires PHP 5.4 or greater.
